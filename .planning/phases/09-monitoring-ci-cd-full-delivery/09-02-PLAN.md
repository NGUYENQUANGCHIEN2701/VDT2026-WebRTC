| Boundary | Description |

| Browser -> nginx :8080 | External entry point serving both static frontend and proxying /api, /ws — untrusted input crosses here (unchanged from prior phases) |
| Prometheus -> backend-1/backend-2 :8080 | Internal-only compose-network scrape path; must never be reachable externally |
| Grafana admin UI :3000 | Exposed on host port 3000 for demo convenience; protected only by GF_SECURITY_ADMIN_PASSWORD |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-09-03 | Elevation of Privilege | Grafana `GF_SECURITY_ADMIN_PASSWORD` | high | mitigate | Sourced from `.env` via `${GRAFANA_ADMIN_PASSWORD:-admin}`, matching the existing `POSTGRES_PASSWORD`/`JWT_SECRET`/`TURN_SECRET` pattern; `.env.example` documents the variable so operators are prompted to set a real value instead of leaving the well-known `admin`/`admin` default in any non-local deployment |
| T-09-04 | Information Disclosure | nginx `location /` SPA fallback ordering | medium | mitigate | `location /` is appended strictly after the existing `/api` and `/ws` prefix locations in `vdt.conf`; nginx's longest-prefix-match means `/api`/`/ws` always take precedence, so static-file serving cannot intercept API/WS traffic — verified via acceptance criteria requiring `/api`/`/ws` blocks remain textually before the new block |
| T-09-05 | Denial of Service | Prometheus/Grafana ports exposed on host (9090, 3000) | low | accept | Acceptable for a local Docker Compose demo environment; no production deployment target for this project per CLAUDE.md scope |

</threat_model>
